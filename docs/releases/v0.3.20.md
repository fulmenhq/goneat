# Goneat v0.3.20 — NOTICE for Distribution

**Release Date**: 2025-12-14
**Status**: Draft

## Overview

Goneat v0.3.20 improves distribution hygiene by adding a top-level `NOTICE` file and ensuring release archives ship `LICENSE` and `NOTICE` alongside the `goneat` binary.

This supports teams (including container distributors) that want consistent attribution artifacts and an easier path to NOTICE tracking.

## ✅ What changed

### NOTICE + packaging

- Added a repository `NOTICE` file.
- Release archives now include:
  - `goneat`
  - `LICENSE`
  - `NOTICE`

### Documentation

- Documented checkmake config override keys supported via `.goneat/assess.yaml`:
  - `lint.make.checkmake.config.max_body_length`
  - `lint.make.checkmake.config.min_phony_targets`

### ADR

- Added ADR-0003 documenting the Linux `CGO_ENABLED=0` musl/glibc compatibility policy.
